# Contribuir a Descifrando la IA

¡Gracias por tu interés! Esta wiki es un proyecto vivo y agradecemos correcciones, referencias nuevas y sugerencias.

## Formas de contribuir

### Correcciones rápidas (typos, enlaces rotos, datos desactualizados)

1. Usa el botón *"Edit this page"* en la parte superior de cualquier módulo del sitio público.
2. GitHub te llevará a un editor que crea automáticamente una rama y un pull request.
3. Describe brevemente el cambio en la descripción del PR.

### Propuestas más grandes (nueva sección, módulo ampliado, estructura)

1. Abre primero un *issue* describiendo la propuesta.
2. Conversamos ahí antes de que inviertas trabajo en un PR.

### Reportar problemas

Abre un *issue* con:
- URL o módulo afectado
- Qué observaste vs. qué esperabas
- Captura de pantalla si es un problema visual

## Flujo editorial

El contenido de este repo se edita primero en un espacio de desarrollo (repo privado) donde se iteran reescrituras RAG-friendly y se mantienen notas internas de producción. Los módulos se liberan aquí **alineados con los episodios del podcast** de [Hijas de Internet](https://instagram.com/hijasdeinternet).

Si tu contribución requiere cambios mayores al proceso editorial, menciónalo en el issue y respondemos con el proceso completo.

## Estilo y tono

- **Español neutro** (no vosotros), accesible, sin jerga innecesaria.
- **Perspectiva crítica**: no antropomorfizamos la IA, cuestionamos quién gana y quién pierde.
- **Referencias**: toda afirmación con dato necesita fuente verificable. Usamos formato de notas al pie (`[^N]`) con URL completa al final del módulo.
- **Latin American context primero**: cuando exista dato regional, úsalo antes que el global.

## Código de conducta

Este proyecto se adhiere al espíritu del [Contributor Covenant](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/): trato respetuoso, inclusivo, sin acoso. Las personas que mantienen el proyecto se reservan el derecho de cerrar PRs o issues que violen ese espíritu.
